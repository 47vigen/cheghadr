# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| latest (`production` branch) | Yes |
| older commits | No |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

To report a vulnerability, use [GitHub's private security advisory](https://github.com/47vigen/cheghadr/security/advisories/new).

Include as much detail as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

We will acknowledge your report within **72 hours** and aim to release a fix within **14 days** for critical issues. You will be credited in the release notes unless you prefer to remain anonymous.
